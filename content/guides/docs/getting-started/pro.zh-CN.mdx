Univer 为了满足更复杂的应用场景，提供了 Univer Pro 版本。Pro 版本包含了更多的功能和特性，适用于需要更高性能和更多功能的应用场景。

文档中与 Univer Pro 相关的部分会以 <IconWrapper className="inline-flex" type="pro" size="sm" icon={null} /> 进行标识。

部分 Univer Pro 的功能需要配合 Univer 的服务端使用，具体请参考 [Univer Pro 服务端](/zh-CN/guides/pro/server) 进行集成。

在未获得许可证的情况下，Univer Pro 的部分功能会受到限制：

| 功能 | 未授权限制 |
| --- | --- |
| 打印 | 水印<br />每次打印不超过 3 页 |
| 协同编辑 | 水印<br />同时协作文档数量不超过 5 个<br />每个文档的协作者数量不超过 3 个 |
| 导入 & 导出 | 水印<br />导入文件不超过 1MB<br />导出文件不超过 10k 单元格 |

关于 Univer Pro 的授权方式，请参考 [Univer Pro 购买页面](https://univer.ai/zh-CN/pricing)。
