## 使用官方主題

Univer 官方提供了兩套主題，分別是 `defaultTheme` 和 `greenTheme`，它們存放於 `@univerjs/themes` 套件中，你可以根據自己的需求選擇其中一個主題。

<Callout>
  當未指定主題時，系統會預設使用 `defaultTheme`。
</Callout>

### 安裝主題插件

#### 預設模式

在預設模式下，主題可以從 `@univerjs/presets` 套件中導入：

```typescript
// import { defaultTheme } from "@univerjs/presets";
import { greenTheme } from '@univerjs/presets'

const { univer } = createUniver({
  theme: greenTheme,
  // theme: defaultTheme,
})
```

#### 插件模式

在插件模式下，主題由 `@univerjs/themes` 套件提供，你需要先安裝該套件。

```package-install
npm install @univerjs/themes
```

```typescript
import { greenTheme } from '@univerjs/themes'
// import { defaultTheme } from "@univerjs/themes";

const univer = new Univer({
  theme: greenTheme,
  // theme: defaultTheme,
})
```

## 自訂主題

只需建立一個符合 `Theme` 介面的物件，即可自訂主題。請參考[這裡](/reference/types/theme)來了解預設主題的變數是如何定義的。

```typescript title="custom-theme.ts"
import type { Theme } from '@univerjs/presets'
// import type { Theme } from "@univerjs/themes";

export const customTheme: Theme = {
  white: '#FEFEFE',
  black: '#1C1C1C',
  primary: {
    50: '#F6F5FF',
    100: '#EDEBFE',
    200: '#DCD7FE',
    300: '#CABFFD',
    400: '#AC94FA',
    500: '#9061F9',
    600: '#7E3AF2',
    700: '#6C2BD9',
    800: '#5521B5',
    900: '#4A1D96',
  },
  // ...
}
```

```typescript
// 在這裡導入你自訂的主題
import { customTheme } from './custom-theme'

const univer = new Univer({
  theme: customTheme,
})
```

## 深色模式

Univer 支援深色模式，你可以透過配置 `darkMode` 屬性為 `true` 來啟用深色模式。

```typescript
new Univer({
  darkMode: true,
})
```

### Facade API

也可以透過 Facade API 動態切換深色模式。

```typescript
// 開啟深色模式
univerAPI.toggleDarkMode(true)

// 關閉深色模式
univerAPI.toggleDarkMode(false)
```
