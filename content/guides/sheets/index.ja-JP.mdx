## Univer Sheets は何ですか

Univer は生産性アプリケーションを構築するためのフルスタックフレームワークで、Univer Sheets は Univer のスプレッドシートコンポーネントです。

Univer Sheets はブラウザとサーバー（Node.js）で実行でき、柔軟で拡張可能かつカスタマイズ可能な特性を持っています。これにより、シンプルな計算機から企業レベルの複雑な財務モデルまで、さまざまなアプリケーションを構築できます。個人開発者でもチームの一員でも、Univer Sheets は強力なスプレッドシートベースのアプリケーションを迅速かつ簡単に構築するのに役立ちます。

## 主な機能

Univer Sheets は現在、幅広い機能をサポートしています：

- **基本スプレッドシート機能**：セル、行、列の作成、編集、削除。
- **数式**：数式を使用してセル内の値を計算。
- **数値形式**
- **データ検証と条件付き書式**
- **フィルタリングと並べ替え**
- **権限管理**
- **画像**
- など...

詳細については、[機能](/guides/sheets/features/core)セクションのページを参照してください。コア機能と主要機能は[GitHubでオープンソース](https://github.com/dream-num/univer)されており、無料で使用できます。

## ドキュメントの使い方

左側にはドキュメントのナビゲーションバーがあります。ドキュメントページは基礎から高度な内容までの順序で整理されており、この順序に従って Univer を段階的に学ぶことができます。または、興味のあるセクションにジャンプすることもできます。サイドバーの上部では Univer Docs のドキュメントを切り替えることもできます。

屏幕の右側には現在のページのアウトラインがあります。ページを素早く見つける必要がある場合は、上部の検索バーまたは検索ショートカット（Ctrl+K または Cmd+K）を使用できます。

我が Univer チームは、[クイックスタート](/guides/sheets/getting-started/quickstart)から始めることをお勧めします。

## 前提知識

Univer のドキュメントは初心者にも理解しやすいように努めていますが、ドキュメントが Univer に焦点を当てるためには、読者がある程度の基礎知識を持っていることを前提としています。

Univer のドキュメントを最大限に活用するためには、JavaScript とそのエコシステムについての基本的な理解が推奨されます。

Node.js 上で Univer を実行することに興味がある場合は、Node.js についての基本的な理解も必要です。

Univer Server の高度な機能を試したり、デプロイしたりする場合は、Linux、Docker、および Docker Compose（または Kubernetes）についての基本的な理解が推奨されます。

## Univer Sheets の拡張

Univer は設計当初から高度に拡張可能であることを念頭に置いていますので、ニーズに応じてカスタマイズできます。[アーキテクチャ](/guides/recipes/architecture/univer)セクションを読んで、Univer Sheets の構造と拡張方法を理解することをお勧めします。また、[CSV インポートプラグインの作成方法](/guides/recipes/practices/csv-import-plugin)から始めることもできます。
