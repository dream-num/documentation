為了滿足更複雜的應用情境，Univer 提供了 Univer Pro 版本。Pro 版本包含更多功能與特性，適用於需要更高效能與更完整能力的場景。

文件中與 Univer Pro 相關的部分會以 <IconWrapper className="inline-flex" type="pro" size="sm" icon={null} /> 進行標示。

部分 Univer Pro 的功能需要搭配 Univer 的伺服端使用，詳情請參考 [Univer Pro 伺服端](/guides/pro/server) 進行整合。

在未取得授權的情況下，Univer Pro 的部分功能將受限：

| 功能 | 未授權限制 |
| --- | --- |
| 進階公式引擎 | 浮水印 |
| 列印 | 浮水印<br />每次列印不超過 3 頁 |
| 圖表 | 浮水印<br />僅支援折線圖、柱狀圖、圓餅圖、條形圖 |
| 迷你圖 | 浮水印 |
| 資料透視表 | 浮水印<br />每個文件僅能建立一個資料透視表 |
| 協同編輯 | 浮水印<br />同時協作的文件數不超過 5 個<br />每個文件的協作者數量不超過 3 人 |
| 編輯紀錄 | 浮水印 |
| 匯入與匯出 | 浮水印<br />匯入檔案不超過 1MB<br />匯出檔案不超過 10k 個儲存格 |

關於 Univer Pro 的授權方式，請參考 [Univer Pro 購買頁面](https://univer.ai/pricing)。
