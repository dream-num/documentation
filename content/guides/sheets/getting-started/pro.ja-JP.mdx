Univer は、より複雑なアプリケーションシナリオに対応するために Pro バージョンを提供しています。Pro バージョンは、より多くの機能と能力を備えており、高いパフォーマンスや多機能性を求めるアプリケーションに最適です。

Univer Pro に関連するドキュメントの部分は、<IconWrapper className="inline-flex" type="pro" size="sm" icon={null} /> でマークされています。

Univer Pro の一部機能は Univer サーバーとの連携が必要です。詳細は [Univer Pro サーバー](/guides/pro/server) をご参照ください。

ライセンス未取得の場合、Univer Pro の一部機能には以下の制限があります：

| 機能 | ライセンス未取得時の制限 |
| --- | --- |
| 高度な数式エンジン | 透かし表示 |
| 印刷 | 透かし表示<br />1回の印刷につき最大3ページまで |
| チャート | 透かし表示<br />折れ線グラフ、棒グラフ、円グラフ、横棒グラフのみ対応 |
| スパークライン | 透かし表示 |
| ピボットテーブル | 透かし表示<br />ドキュメントごとに1つのみ作成可能 |
| 共同編集 | 透かし表示<br />同時に共同編集できるドキュメントは最大5つまで<br />1ドキュメントあたり最大3人まで |
| 編集履歴 | 透かし表示 |
| インポート & エクスポート | 透かし表示<br />インポート可能なファイルサイズは1MBまで<br />エクスポート可能なセル数は最大10,000セルまで |

Univer Pro のライセンス取得方法については、[Univer Pro 購入ページ](https://univer.ai/en-US/pricing) をご覧ください。
