## IWorksheetData

[`IWorksheetData`](https://reference.univer.ai/en-US/interfaces/IWorksheetData) は Univer Sheet におけるワークシートのスナップショットを表します。

### プロパティ

| プロパティ | タイプ | 説明 |
|----------|------|-------------|
| id | `string` | ワークシートの一意識別子。 |
| name | `string` | ワークシートの名前。 |
| tabColor | `string` | ワークシートタブの色。 |
| hidden | `BooleanNumber` | シートが非表示かどうか。デフォルト: `BooleanNumber.FALSE`。 |
| freeze | `IFreeze` | フリーズペインの設定。 |
| rowCount | `number` | 行の総数。 |
| columnCount | `number` | 列の総数。 |
| defaultColumnWidth | `number` | 列のデフォルト幅、単位 `px`。 |
| defaultRowHeight | `number` | 行のデフォルト高さ、単位 `px`。 |
| mergeData | `IRange[]` | 結合セル範囲の配列。 |
| cellData | `IObjectMatrixPrimitiveType<ICellData>` | セル内容のマトリックス。[詳細情報。](/guides/sheets/model/cell-data) |
| rowData | `IObjectArrayPrimitiveType<Partial<IRowData>>` | 行データオブジェクトの配列。 |
| columnData | `IObjectArrayPrimitiveType<Partial<IColumnData>>` | 列データオブジェクトの配列。 |
| rowHeader | `{ width: number; hidden?: BooleanNumber; }` | 行ヘッダーの設定。 |
| columnHeader | `{ height: number; hidden?: BooleanNumber; }` | 列ヘッダーの設定。 |
| showGridlines | `BooleanNumber` | グリッド線が表示されるかどうか。 |
| rightToLeft | `BooleanNumber` | ワークシートが右から左モードかどうか。 |
| [defaultStyle](/guides/sheets/features/core/default-style)? | `Nullable<IStyleData>` | ワークシートのデフォルトスタイル |

### 例

```typescript
const worksheetData: IWorksheetData = {
  id: 'sheet1',
  name: 'Sheet 1',
  tabColor: '#FF0000',
  hidden: BooleanNumber.FALSE,
  freeze: { xSplit: 1, ySplit: 1, startRow: 1, startColumn: 1 },
  rowCount: 1000,
  columnCount: 26,
  defaultColumnWidth: 100,
  defaultRowHeight: 25,
  mergeData: [],
  cellData: {
    0: {
      0: {
        v: 123,
      },
    },
  },
  rowData: [],
  columnData: [],
  rowHeader: { width: 40 },
  columnHeader: { height: 20 },
  showGridlines: BooleanNumber.TRUE,
  rightToLeft: BooleanNumber.FALSE,
}
```

この構造は、レイアウト、内容、表示設定を含むワークシートデータの包括的な表現を提供します。
