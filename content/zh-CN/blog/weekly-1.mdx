# Univer 周报 #1

欢迎来到 **Univer 周报**！在这个全新系列中，我们将每周为你带来 Univer 的最新动态，分享项目的进展、亮点和更新。希望通过这些内容，帮助你更好地了解、使用，并积极参与 Univer 的开发与成长！

## Univer SDK 新版本发布

上周末我们发布了 OSS 和 Pro SDK 的 0.5.2 版本，新增了迷你图功能，并修复了一些 bug，你可以在我们的[发布日志](https://www.univer.ai/zh-CN/changelog)了解全部的变更内容。目前 Univer Sheet 已经基本实现了电子表格的主要功能，欢迎大家尝试使用并向我们提供反馈，我们根据大家的需求会在接下来的版本中持续完善电子表格的功能，同时也会逐步推出更多的组件。

## Facade API 持续优化中

最近一段时间 Univer 团队花费了许多时间提升 Facade API 的能力以提升 Univer 的开发体验，我们已经在这方面取得了一些进展。我们将从 0.5.2 版本开始逐步发布这些改进，希望能够帮助开发者更好地使用 Univer。

## React 19 兼容性

本月初 React 发布了 19.0.0 版本。Univer 一般使用上游依赖的最新版本，然而有部分用户仍在使用 React 17 之前的版本，React 19 对比这些版本存在很多破坏性更新，因此本周我们开始了对多版本 React 的兼容工作，目标是为停留在 React 不同版本的用户提供无痛的接入体验。这项工作预计在下周完成。

尽管 Univer 会尽力支持多个 React 版本，但我们仍强烈建议您定期升级您的 React（以及其他主流依赖）的版本，以和社区更好地兼容。

## 协同稳定性

我们一直致力于为我们的商业版用户提供一流的协同编辑体验。本月我们完成了一次系统性的压力测试，识别了典型场景下影响协同系统的稳定性的因素，在接下来的几周中我们会逐步解决这些问题。
