# Univer Go 简介

欢迎使用 Univer Go 文档！

## Univer Go 是什么

Univer Go 是一款功能强大的脚本编写与执行工具，适用于跨平台环境，帮助开发者创建和运行自动化脚本、数据库连接、以及自定义应用。通过简洁的界面和高效的操作流程，Univer Go 提供了易于使用的编程环境，适合无论是初学者还是专业开发者使用。

## 主要功能

Univer Go 现在支持多种功能，包括：

- **跨平台支持**：Univer Go 提供 macOS 和 Windows 版本，用户可以轻松下载并安装，支持在两种操作系统上无缝运行。
- **注册与登录**：通过电子邮箱或 Google 账号快捷登录，支持云端同步和管理。
- **工作区管理**：创建和管理多个工作区（Workspace），支持创建、编辑、删除工作区，并可以在不同的工作区间切换。
- **脚本创建与执行**：在工作区内创建并编辑自定义脚本，包括客户端脚本和服务器端脚本，支持与 AI 代理、数据库等外部系统交互。
- **脚本运行与预览**：提供多种触发器（如 onOpen、onCreate、onClose）执行脚本，运行时可以选择在电子表格中查看效果，并支持代码的实时预览与调试。
- **文档管理**：工作区内创建和删除文档，文档的管理与访问也十分便捷，删除工作区时，文档会自动删除，确保文件管理的整洁。
