# 网格线 <VersionBadge version="0.4.2+" />

<FeatureMeta texts={['✅', '-', '-', '-', 'UniverSheetsCorePreset' ]} />

从 0.4.2 版本开始，你可以按子表切换网格线的显示和隐藏。

## Facade API

### 切换子表网格线的显示和隐藏

```typescript
const fWorkSheet = univerAPI.getActiveWorkbook().getActiveSheet();

// 隐藏
fWorkSheet.setHiddenGridlines(true); 

// 显示
fWorkSheet.setHiddenGridlines(false);
```

### 获取当前子表网格线的隐藏状态

```typescript
const fWorkSheet = univerAPI.getActiveWorkbook().getActiveSheet();

// 检查网格线是否隐藏
if (fWorkSheet.hasHiddenGridLines()) {
  console.log('网格线已隐藏');
}
```

### 设置网格线颜色

```typescript
const fWorkSheet = univerAPI.getActiveWorkbook().getActiveSheet();

// 设置网格线颜色为红色
fWorkSheet.setGridLinesColor('#ff0000');
console.log(fWorkSheet.getGridLinesColor()); // #ff0000
```
