# 高级功能简介

Univer 的核心框架以及许多基础功能[在 GitHub 上开源](https://github.com/dream-num/univer)，并基于 Apache-2.0 授权协议允许个人与组织免费使用，同时我们为企业用户提供了一系列高级功能，包括：

- 数据透视表
- 打印
- 导入导出
- 协同编辑（包含协同光标和历史记录）
- 协同浏览 Live Share

等等，这些功能将会在文档中以 <span className="mr-2"><ProTag /></span> 标识。

高级功能基于 Univer 商业授权协议，可以免费使用，不过存在一些限制[^1]。如果你需要解锁限制，可以联系我们的销售团队以购买许可证。你可以在 [许可证](/guides/sheets/pro-features/license) 一章了解你是否需要许可证，以及如何获取和使用许可证。

## 集成 Univer 高级功能 [#integrate-advanced-features]

<Steps>
  ### 安装并运行服务端

  ```shell
  bash -c "$(curl -fsSL https://get.univer.ai)"
  ```

  ### 安装并运行客户端 [#install-and-run-the-frontend]

  <Tabs items={['npm', 'pnpm', '从单 HTML 文件开始']}>
    <Tabs.Tab>
      执行以下命令需要设备安装有 `nodejs >= 18`, `npm >= 8` 和 `git`。

      ```shell
      npx degit dream-num/univer-pro-sheet-start-kit univer-pro-sheet-start-kit
      cd univer-pro-sheet-start-kit
      npm install
      npm run dev
      ```
      <Callout emoji="⚠️">
        如果服务端部署在其他机器上，需要在 `vite.config.ts` 中把 `server` 配置里的 `localhost` 改为该机器的 IP 地址。
      </Callout>
      🎉 恭喜你，你已成功创建了包含了服务端的 Univer 项目，你可以访问 [http://localhost:5173/](http://localhost:5173/) 体验高级功能和协同编辑。

      你可以在 [dream-num/univer-pro-sheet-start-kit](https://github.com/dream-num/univer-pro-sheet-start-kit) 找到该模板的源代码。     
    </Tabs.Tab>
    <Tabs.Tab>
      执行以下命令需要设备安装有 `nodejs >= 18`, `pnpm >= 7` 和 `git`。

      ```shell
      pnpm dlx degit dream-num/univer-pro-sheet-start-kit univer-pro-sheet-start-kit

      ```

        如果服务端部署在其他机器上，需要在 `vite.config.ts` 中把 `server` 配置里的 `localhost` 改为该机器的 IP 地址。
      </Callout>

      🎉 恭喜你，你已成功创建了包含了服务端的 Univer 项目，你可以访问 [http://localhost:5173/](http://localhost:5173/) 体验高级功能和协同编辑。

## Univer 服务端

在前面的例子中我们已经接入 Univer 的服务端（Univer Server），这些功能包括：

* 导入导出
* 协同编辑
* Live Share

请参考 [服务概述](/guides/sheets/pro-features/server/overview) 以进一步了解 Univer 服务端。
