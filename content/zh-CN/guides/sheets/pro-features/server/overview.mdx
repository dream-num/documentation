# 服务概述

出于安全、性能等目的的考虑，部分 Univer 的功能在后端服务中实现。如果你使用到了以下任一功能，需要部署univer服务端方能支持。

- 打印
- 导入 & 导出
- 多人协作编辑
- Live Share

Univer 提供的服务端不仅能满足你的上述功能需求，还支持与你的自有系统进行集成以实现更多的能力。比如：

- 用户身份认证，只有你系统中的用户才能访问 Univer 服务
- 权限管理，管控哪些用户/用户组能编辑、阅读、导出 等等
- 用户文件夹管理
- 通知，你可以监听 Univer 发出的事件，当用户在文档评论中@了某人，你可以将通知发送到被@用户的邮箱/IM
